# CSS Framework Proposal

## Color

### Color Class

```css
.class-name {
  color: var(--color-name);
}

.class-name-1 {
  color: var(--color-name-1);
}
```

## Width

```css
/* Percent */
.w-1 {
  width: 1%;
}
/* ... */
.w-100 {
  width: 1%;
}
/* ViewPort */
.vw-1 {
  width: 1vw;
}
/* ... */
.vw-100 {
  width: 100vw;
}

/* Pixel */
.w-1px {
  width: 1px;
}
/* ... */
.w-600px {
  width: 600px;
}
```
